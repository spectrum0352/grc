# Solution Delivery Lead – Cyber Security

## Position Summary

The Solution Delivery Lead is responsible for leading the design, implementation, optimization, and operational support of enterprise cybersecurity solutions across cloud, endpoint, identity, data protection, and security operations domains. The role involves managing security transformation programs, providing technical leadership, supporting incident response activities, and ensuring alignment with security frameworks and regulatory requirements.

## Key Responsibilities

### Security Strategy and Solution Delivery

* Lead the design, implementation, and optimization of enterprise security solutions across cloud, endpoint, identity, data protection, and security monitoring platforms.
* Act as a trusted advisor to stakeholders on cybersecurity architecture, security controls, and risk mitigation strategies.
* Manage end-to-end delivery of security projects, including assessment, design, implementation, testing, deployment, and transition to operations.
* Develop security roadmaps and transformation strategies aligned with business objectives and security requirements.

### Cloud and Microsoft Security

* Lead deployment and management of Microsoft security technologies, including:

  * Entra ID (Azure AD)
  * Microsoft Defender Suite
  * Microsoft Sentinel
  * Microsoft Intune
  * Microsoft Purview
  * Security Copilot
  * Exchange Online Protection
  * SharePoint Online
  * Teams
  * OneDrive for Business
* Support migration and consolidation initiatives involving security tools and platforms.
* Design and implement security controls such as Multi-Factor Authentication (MFA), Conditional Access, Data Loss Prevention (DLP), Information Protection, and Identity Governance.

### Security Assessments and Risk Management

* Conduct security assessments, configuration reviews, and technical health checks.
* Evaluate security posture against industry frameworks and standards such as:

  * NIST
  * CIS Controls
  * ISO 27001
  * PCI DSS
  * CISA Guidelines
* Identify security gaps and provide remediation recommendations.
* Support development and implementation of security policies, standards, and procedures.

### Endpoint Security and Detection & Response

* Manage and support enterprise endpoint security platforms and Endpoint Detection and Response (EDR) solutions.
* Perform endpoint security architecture reviews and design assessments.
* Configure and maintain endpoint protection policies, detection rules, exclusions, and monitoring capabilities.
* Develop detection use cases and security controls aligned with frameworks such as MITRE ATT&CK.
* Monitor endpoint security posture and drive continuous improvement initiatives.

### Security Operations and Incident Response

* Support investigation, triage, containment, and remediation of security incidents.
* Collaborate with SOC, Threat Intelligence, Infrastructure, and Engineering teams during security investigations.
* Analyze malware activity, endpoint threats, account compromise events, and policy violations.
* Provide advanced troubleshooting and escalation support for complex security incidents and service disruptions.
* Develop operational runbooks, standard operating procedures (SOPs), and incident response documentation.

### Security Monitoring and Engineering

* Design and maintain security dashboards, reporting frameworks, and operational metrics.
* Develop and optimize security monitoring, detection, and alerting capabilities.
* Perform security engineering activities to address emerging threats and operational challenges.
* Review system security architecture, network diagrams, vulnerability findings, and patch management requirements.

### Stakeholder Management and Leadership

* Serve as the primary technical lead and escalation point for security initiatives.
* Coordinate with business, infrastructure, security, and vendor teams to deliver security solutions.
* Present technical findings and recommendations to leadership and stakeholders.
* Mentor junior engineers and consultants and provide technical guidance across teams.
* Support presales activities, solution proposals, and technical workshops when required.

### Governance, Reporting, and Compliance

* Develop executive and technical security reports.
* Define and track key risk indicators (KRIs) and key performance indicators (KPIs).
* Maintain accurate documentation of security assessments, incidents, configurations, and remediation activities.
* Ensure compliance with internal policies, security standards, and regulatory requirements.
